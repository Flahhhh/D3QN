# D3QN
Here you can find a D3QN (Duelling Double DQN with prioritized experience replay) algorithm implementation for the Gymnasium CartPole-v1 environment.

To train, run:

  pip install -r requirements.txt
  python train.py

To test the model, run:
  python test.py
